<form>
<orth type="lemma">måtte</orth>
<em><span class="dic_color"><span class="dic_comment">pres må, pret måtte</em>
модальный глагол, входящий в состав модального сказуемого, при этом последующий смысловой глагол употребляется без частицы å:
</form>
<sense>
выражает долженствование, приказание, категорическое требование:
<сit type="example"><text>det må skje</text><cit type ="ex_translation"><language><idno type="iso639-3">rus</idno></language><text>это должно случиться</text></cit></cit>
<сit type="example"><text>du må ikke skrive det</text><cit type ="ex_translation"><language><idno type="iso639-3">rus</idno></language><text>ты не должен этого писать</text></cit></cit>
<сit type="example"><text>han må ikke få høre noe om det</text><cit type ="ex_translation"><language><idno type="iso639-3">rus</idno></language><text>об этом он не должен слышать (знать)</text></cit></cit>
<сit type="example"><text>han må skynde seg</text><cit type ="ex_translation"><language><idno type="iso639-3">rus</idno></language><text>он должен торопиться</text></cit></cit>
</sense>
<sense>
указывает на разрешение, возможность:
<сit type="example"><text>du må gjerne bli sittende</text><cit type ="ex_translation"><language><idno type="iso639-3">rus</idno></language><text>ты можешь сидеть</text></cit></cit>
<сit type="example"><text>må jeg spørre?</text><cit type ="ex_translation"><language><idno type="iso639-3">rus</idno></language><text>можно ли мне спросить?</text></cit></cit>
<сit type="example"><text>om jeg må si så</text><cit type ="ex_translation"><language><idno type="iso639-3">rus</idno></language><text>если мне можно так сказать</text></cit></cit>
</sense>
<sense>
указывает на предположение, вероятность, обычно употребляется форма måtte:
<сit type="example"><text>det måtte være Deres venn</text><cit type ="ex_translation"><language><idno type="iso639-3">rus</idno></language><text>это, должно быть, был ваш приятель</text></cit></cit>
<сit type="example"><text>det må være så</text><cit type ="ex_translation"><language><idno type="iso639-3">rus</idno></language><text>должно быть, это так</text></cit></cit>
</sense>
<sense>
в повелительном наклонении выражает пожелание:
<сit type="example"><text>måtte du ha lykken med!</text><cit type ="ex_translation"><language><idno type="iso639-3">rus</idno></language><text>пусть счастье тебе улыбнётся!, желаю тебе счастья!</text></cit></cit>
