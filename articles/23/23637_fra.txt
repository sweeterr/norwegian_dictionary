<orth type="lemma">fra</orth>
<u>prep</u>
</form>
<sense>
в пространственном значении указывает на удаление от исходного пункта движения, от, из, с:
<сit type="example"><text>fra Norge</text><cit type ="ex_translation"><language><idno type="iso639-3">rus</idno></language><text>из Норвегии</text></cit></cit>
<сit type="example"><text>fra byen</text><cit type ="ex_translation"><language><idno type="iso639-3">rus</idno></language><text>из города</text></cit></cit>
<сit type="example"><text>fra øst</text><cit type ="ex_translation"><language><idno type="iso639-3">rus</idno></language><text>с востока</text></cit></cit>
<сit type="example"><text>fra borde</text><cit type ="ex_translation"><language><idno type="iso639-3">rus</idno></language><text>из-за стола</text></cit></cit>
</sense>
<sense>
в пространственном значении указывает на расстояние от какого-л. предмета
<сit type="example"><text>Kristiansund ligger 120 kilometer fra Ålesund</text><cit type ="ex_translation"><language><idno type="iso639-3">rus</idno></language><text>Кристиансунн находится в 120 километрах от Олесунна</text></cit></cit>
<сit type="example"><text>fra ... til ...</text><cit type ="ex_translation"><language><idno type="iso639-3">rus</idno></language><text>от ... до ...; <lbl>перен.</lbl> начиная ... и кончая ...</text></cit></cit>
</sense>
<sense>
во временном значении указывает на удаление от определённого момента, с, от:
<сit type="example"><text>fra denne tid</text><cit type ="ex_translation"><language><idno type="iso639-3">rus</idno></language><text>с этого времени, с этих пор</text></cit></cit>
<сit type="example"><text>fra gutter av</text><cit type ="ex_translation"><language><idno type="iso639-3">rus</idno></language><text>с детства, с того времени, как они (мы, вы) были мальчиками</text></cit></cit>
<сit type="example"><text>fra klokka tre til sju</text><cit type ="ex_translation"><language><idno type="iso639-3">rus</idno></language><text>от трёх до семи часов</text></cit></cit>
<сit type="example"><text>venner fra skolen</text><cit type ="ex_translation"><language><idno type="iso639-3">rus</idno></language><text>друзья со школьной скамьи</text></cit></cit>
<сit type="example"><text>fra før</text><cit type ="ex_translation"><language><idno type="iso639-3">rus</idno></language><text>от прежних времён, ещё прежде, ещё до этого</text></cit></cit>
</sense>
<sense>
указывает на происхождение, на появление из какого-л. источника, от, из:
<сit type="example"><text>hvor er han fra?</text><cit type ="ex_translation"><language><idno type="iso639-3">rus</idno></language><text>откуда он родом?</text></cit></cit>
<сit type="example"><text>han nedstammer fra</text><cit type ="ex_translation"><language><idno type="iso639-3">rus</idno></language><text>он происходит от (такого-то)</text></cit></cit>
<сit type="example"><text>ordet er lånt fra fransk</text><cit type ="ex_translation"><language><idno type="iso639-3">rus</idno></language><text>это слово заимствовано из французского языка</text></cit></cit>
</sense>
<sense>
указывает на отделение от чего-л., освобождение от чего-л., от, у:
<сit type="example"><text>skyve noe fra seg</text><cit type ="ex_translation"><language><idno type="iso639-3">rus</idno></language><text>оттолкнуть (отпихнуть) от себя</text></cit></cit>
<сit type="example"><text>fra hinannen</text><cit type ="ex_translation"><language><idno type="iso639-3">rus</idno></language><text>один от другого</text></cit></cit>
<сit type="example"><text>avsette en fra sin stilling</text><cit type ="ex_translation"><language><idno type="iso639-3">rus</idno></language><text>освободить кого-л. от должности</text></cit></cit>
<сit type="example"><text>være fra seg</text><cit type ="ex_translation"><language><idno type="iso639-3">rus</idno></language><text>быть вне себя</text></cit></cit>
</sense>
