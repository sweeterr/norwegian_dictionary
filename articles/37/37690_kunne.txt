<orth type="lemma">kunne</orth>
<div><em><span class="dic_color"><span class="dic_comment">pres kan, pret kunne, p.p. kunnet</em>

<sense> мочь, быть в состоянии
<div style="margin-left:5px"><strong><p><span class="dic_example">ikke kunne mere — не мочь
</sense>
<sense> знать
</sense>
<sense> уметь
</sense>
<sense> служит для выражения действия, передающего привычку, обыкновение:
<div style="margin-left:5px"><strong><p><span class="dic_example">slik kunne hun ligge i timevis — так она могла лежать (лежала) часами
</sense>
<sense> служит для выражения возможности, вероятности, допустимости совершения действия:
<div style="margin-left:5px"><strong><p><span class="dic_example">det kan hende — может случиться
<div style="margin-left:5px"><strong><p><span class="dic_example">det kan tenkes — можно подумать
</sense>
<sense> в форме прошедшего времени употребляется для выражения гипотетического действия пожелания, отнесённого к настоящему моменту:
<div style="margin-left:5px"><strong><p><span class="dic_example">han kunne gjerne se deg — он, кажется (вероятно), увидит тебя
</sense>
<sense> служит для выражения разрешения сделать что-л.:
<div style="margin-left:5px"><strong><p><span class="dic_example">de kan gjøre som de vil — они могут поступать, как им угодно
<div style="margin-left:5px"><strong><p><span class="dic_example">du kan ta denne røde blyanten — ты можешь взять этот красный карандаш
</sense>
<sense> служит для выражения угрозы:
<div style="margin-left:5px"><strong><p><span class="dic_example">du kan forsøke (prøve våge) — попробуй (посмей) только
<div style="margin-left:5px"><strong><p><span class="dic_example">kunne med én (noe) — ладить с кем-л. (чем-л.); выносить кого-л. (что-л.)
</sense>
